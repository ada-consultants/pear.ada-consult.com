ADA Consultants PEAR channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover pear.ada-consult.com
Listing available packages: 
pear remote-list -c ada
Installing a package: 
pear install ada/package_name
Installing a specific version/stability: 
pear install ada/package_name-1.0.0
pear install ada/package_name-beta
Receiving updates via a feed: 
http://pear.ada-consult.com/feed.xml
